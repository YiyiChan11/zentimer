# ZenTimer 使用文档

本文档面向**最终用户**，介绍 ZenTimer（禅意番茄钟）的各项功能与操作方法。
开发者相关请见 [README.md](README.md) 与 [CONTRIBUTING.md](CONTRIBUTING.md)。

---

## 1. 安装与启动

### Windows 桌面端
- 从下载页或发布渠道获取 `ZenTimer_x64-setup.exe`，双击安装。
- 首次安装后，开始菜单 / 桌面会出现 ZenTimer 图标。
- 启动后在系统托盘（右下角）会出现 ZenTimer 图标，可右键进行「显示 / 隐藏 / 退出」。

> 若 Windows SmartScreen 提示“未知发布者”，这是未购买代码签名证书所致，点击「仍要运行」即可。

### Web 版（PWA）
- 用 Chrome / Edge 打开 Web 版地址，点击地址栏右侧的「安装」图标，即可像原生应用一样添加到桌面。
- Web 版功能与桌面端基本一致，但悬浮窗在浏览器中使用的是 Document Picture-in-Picture 形式。

## 2. 主界面

| 区域 | 说明 |
|------|------|
| 中央圆环 | 倒计时显示，环的进度表示剩余时间；暂停时荧光会变暗 |
| 时间选择区 | 设置本次专注时长（固定 / 随机） |
| 底部三按键 | 开始专注 / 暂停·继续 / 重置 / 跳过 |
| 右上角齿轮 | 打开设置面板 |

---

## 3. 设置专注时间

### 固定时长
- 拖动滑块选择 5–90 分钟（5 分钟一档），滑块经过常用档位（如 25 / 45 / 60）会自动吸附。
- 选择后点击「Start Focus」开始专注。

### 随机范围
- 切换到“随机”模式，设定一个时间区间（如 25–50 分钟）。
- 每次开始专注时，系统会在该区间内随机选一个时长，避免机械感。

---

## 4. 专注 / 休息模式

- **专注（Focus）**：圆环呈琥珀色发光，倒计时进行中。
- **暂停**：点击暂停后，圆环荧光平滑变暗至约 20% 亮度，与进行中状态形成明显区分；再次点击继续。
- **休息（Break）**：专注结束后自动进入休息，圆环转为青绿色，默认 10 分钟（可在设置中调整）。
- **微休息（Buffer）**：专注过程中会随机触发一次 15 秒的短暂微休息（提示“喝口水、闭眼放松”），随后自动继续。可在设置中关闭。

## 5. 悬浮窗

悬浮窗是一个常驻桌面、置顶的小计时器，适合在看书 / 写代码时放在屏幕一角。

### 开关
- 打开 **设置面板 → 悬浮窗 → Open Floating** 开启；再次点击 **Close Floating** 关闭。

### 操作手势（桌面端）
| 操作 | 效果 |
|------|------|
| 单击悬浮窗 | 暂停 / 继续计时 |
| 双击悬浮窗 | 打开主窗口 |
| 长按 / 拖动 | 移动悬浮窗位置 |
| 右上角 × | 关闭悬浮窗 |

### 透明度
- 设置面板中的「悬浮窗透明度」滑块控制整体可见度：
  - **0%**：悬浮窗若隐若现（约 5% 可见），主要能看到背后的内容，时间数字仍清晰可读。
  - **100%**：悬浮窗完全实心，背后内容被遮挡。
- 文字亮度始终比窗体形状高约 5–10%，保证低透明度下也能看清时间。

### 锁定 / 解锁 ⭐ 新增
- **锁定**：在设置面板中点击「锁定悬浮窗」（位于 Open/Close Floating 下方、透明度上方）。锁定后悬浮窗**无法被点击、拖动、选中**，避免误触；关闭也需在软件内完成。
- **解锁**：再次点击同一按钮即可解锁，恢复正常交互。
- 悬浮窗右下角也提供锁定 / 解锁按钮（与关闭按钮同尺寸、同位置排布），开启悬浮窗后显示在关闭按钮上方。

## 6. 设置面板

打开右上角齿轮进入，包含：

- **休息时长**：每次专注后的休息分钟数。
- **微休息**：开启 / 关闭随机微休息，及触发参数。
- **音量**：提示音大小，含试听；采用对数曲线，低音量也细腻。
- **自动开始**：专注 / 休息结束后是否自动进入下一阶段。
- **语言**：中文 / English 切换（全应用即时生效）。
- **悬浮窗**：开关、透明度、锁定 / 解锁。
- **下载**：查看各平台版本信息。
- **更新**：点击「Check for update」检查并下载新版本；下载进度显示在面板内。

| `Space` | 开始 / 暂停 |
| `Esc` | 重置当前计时 |

## 8. 自动更新

- 桌面端内置更新器。当发布新版本后，在设置 → 更新 中点击「Check for update」即可下载安装。
- 下载过程中，设置面板内会显示进度条；完成后按提示重启应用。
- 更新源托管在 Gitee，安装包经 minisign 签名校验，确保完整与安全。

---

## 9. 多语言

- 在设置面板底部切换「中文 / English」。
- 切换后界面文字（含“今日 N 轮”等）即时变更，无需重启。

## 10. 常见问题（FAQ）

**Q：悬浮窗拖不动 / 点不动？**
A：可能已被锁定。请到设置面板点击「解锁悬浮窗」，或使用悬浮窗右下角的解锁按钮。

**Q：低透明度下几乎看不到悬浮窗？**
A：这是设计预期——0% 时仅约 5% 可见，主要让背后内容透出；时间文字保持更高亮度以便阅读。调高透明度滑块即可更明显。

**Q：暂停后为什么圆环变暗？**
A：为区分“进行中”与“已暂停”状态，暂停时荧光刻意调暗。继续后会恢复明亮。

**Q：更新检查没反应 / 报已是最新？**
A：请确认已安装版本低于最新发布版本；旧版本（如 1.1.2 之前）指向旧更新源，需手动安装一次新版本以启用 Gitee 自更新。

**Q：Android 版在哪？**
A：Android 端仍在规划中（下载页显示 “coming soon”）。当前支持 Web 与 Windows 桌面端。
